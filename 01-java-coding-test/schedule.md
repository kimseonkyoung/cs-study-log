# 예상 일정 (2026-07-09 산출, 07-09 세부 강의 단위로 개정)

> **주차는 데드라인이 아니라 예상치다.** 실제 완료 기준은 각 섹션 `notes.md`의
> 강의 체크리스트 + "문제풀이 회고"가 채워지는 것이다. 밀려도 조급해하지 않는다.

## 진행 단위 (개정)

섹션을 통째로 몰아 듣지 않고, **세부 강의 1~2개 단위**로 사이클을 돈다:

1. 세부 강의 1개 시청 (같은 개념이면 2개까지 묶어도 됨 — 예: 회문문자열 + 팰린드롬)
2. 해당 강의의 개념을 `notes.md`에 정리 + 강의 문제 직접 구현
3. 그 개념과 관련된 프로그래머스 문제 풀이 (`solutions/`)

- 회사 세션: 강의 시청 + 개념 정리 + 강의 문제 구현 (폐쇄망, 자바 가능)
- 집 세션: 관련 프로그래머스 문제 풀이 + 회고

# 코딩테스트 자바 커리큘럼 — 강의 순서 가이드

기준 강의: 자바 알고리즘 강의(섹션 2~11, 총 11개 섹션)
주 4일 가능(회사 2일 개념 / 집 2일 문제풀이), 회당 2시간

**진행 방식: 주차 데드라인이 아니라 섹션 완료 기준으로 진행한다.**
- 강의 하나 듣고 → 개념정리(notes.md) → 관련 문제풀이(solutions/) → 다음 섹션
- 한 섹션의 notes.md 체크리스트가 다 체크되고 "문제풀이 회고"까지 채워지면 완료로 간주
- 진도가 빠르면 한 세션에 여러 섹션 진행, 느리면 한 섹션을 여러 세션에 걸쳐 진행해도 됨

## 진행 원칙
- 개념정리(`notes.md`)와 문제풀이(`solutions/`)는 항상 분리해서 관리
- 강의 분량이 회사 2시간을 넘으면(3, 4주차처럼) 집 세션 시작 전에 남은 영상 마저 보고 문제풀이로 전환해도 되고, 다음 회사 세션으로 이어도 됨

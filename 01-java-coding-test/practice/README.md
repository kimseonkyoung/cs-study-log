# practice — 템플릿 연습장

외운 템플릿을 **안 보고** 쳐보는 스크래치 폴더.
여기 있는 `.java` 파일은 커밋되지 않는다 (연습 코드는 흔적 안 남김).
남기고 싶은 코드가 생기면 해당 섹션의 `solutions/`로 옮긴다.

## 사용법

1. 아무 이름으로 파일 만들어서 템플릿을 기억만으로 작성 (예: `scratch.java`)
2. `java scratch.java`로 바로 실행해서 결과 확인 (JDK 11+ 단일 파일 실행)
3. 막히면 그때 `00-intro/notes.md` 열어서 확인 → 틀린 부분 표시 → 다음날 다시

## 현재 연습 목록

- [ ] **버블 정렬** — 루프 범위 `i < n-1`, `j < n-1-i` / 스왑 3줄 / swapped 조기 종료
- [ ] **이분탐색** — `left <= right` (등호!) / `mid+1`, `mid-1` / 못 찾으면 -1
- [ ] **문자열 뒤집기** — StringBuilder append, 반복문에서 String += 금지

## 자가 체크 (템플릿 쳤으면 이것부터 확인)

- 버블정렬: `{1}` 한 개짜리, 이미 정렬된 배열 넣어도 안 터지는가?
- 이분탐색: 맨 앞/맨 뒤/없는 값/빈 배열 4개 케이스 다 통과하는가?
- 3일 연속 안 보고 성공하면 체크 표시하고 다음 템플릿 추가

# 복습 체크포인트

섹션을 계속 새로 나가다 보면 앞에서 배운 게 잊혀지기 쉽다.
"섹션 하나 끝날 때마다 살짝 뒤돌아보기"를 규칙으로 둔다.

## 규칙

**섹션 2개 끝날 때마다 복습 체크포인트 1회**
(예: 02-string, 03-array 끝나면 → 체크포인트 → 04, 05 끝나면 → 체크포인트 ...)

체크포인트에서 하는 것:
1. **개념 퀴즈 3문제** — 지금까지 끝낸 섹션들의 notes.md 기반으로 즉석 질문
   (예: "StringBuilder가 왜 빠른지 설명해봐", "DFS와 BFS 언제 각각 쓰는지" 등)
2. **재풀이 1문제** — 예전에 풀었던 문제 중 하나를 다시, 코드 안 보고 풀어보기
   (막히면 그건 아직 완전히 내 것이 안 된 것 → 해당 섹션 notes.md 다시 훑기)
3. **막혔던 것 다시 보기** — solutions/ 폴더의 "막혔던 점" 주석들만 훑어보기

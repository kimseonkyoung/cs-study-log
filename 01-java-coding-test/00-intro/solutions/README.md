# Week 1 — 프로그래머스 문제풀이

`notes.md`에서 정리한 개념을 바탕으로 실제로 푼 문제를 여기에 파일 하나씩 쌓는다.

파일명 규칙: `lv{레벨}-{문제이름}.java` (예: `lv1-이상한문자만들기.java`)

각 파일 상단에는 아래 형식으로 주석을 남긴다.
```java
/**
 * 문제: (문제 요약 한두 줄)
 * 접근: (어떤 방식으로 풀었는지)
 * 시간복잡도: O(?)
 * 막혔던 점: (있었다면)
 */
```
